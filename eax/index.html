Examples of slice

25 Jan 2016

The following are some examples that helped me to better understand slice. Remember that slice makes a copy of the original array, so the original array is not mutated.

The parameters are as follows:
array.slice(startIndex, endIndex)

// Given an array of integers from 0 to 5
> var arr = [0, 1, 2, 3, 4, 5]
// Without the optional second parameter, a copy of an array is returned with the given index until the end of the array
> arr.slice(1) // [1, 2, 3, 4, 5]
// Again, the original array remains the same
> arr // [0, 1, 2, 3, 4, 5]
// If you include a second argument, it will slice from the first argument's index to the second argument up to but not including the element at that index
> arr.slice(1, 3) // [1, 2]
> arr // [0, 1, 2, 3, 4, 5]
// Using a negative number as an integer will return the number of items from the end of the array
> arr.slice(-1) // [5]
> arr // [0, 1, 2, 3, 4, 5]
> arr.slice(-3) // [3, 4, 5]
> arr // [0, 1, 2, 3, 4, 5]

When writing functions, there is a way to use slice that is very useful.
Array.prototype.slice.call()

When writing a function, arguments can be iterated over similar to an array and have a length property but are not considered and array, rather an array-like object. As you can see in the example below, we use Array.prototype.slice.call(arguments) to create an array of the arguments. Now you can use the native array methods that would otherwise not have been available on the array-like arguments. 

var example = function() {  
  var argumentsArray = Array.prototype.slice.call(arguments);
  console.log(argumentsArray);
  return argumentsArray.pop();
}

> example(1, 2, 3) // [1, 2, 3]
